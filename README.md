# Reconstruction Studies
Compilation of 3D reconstruction methods

## Dust3r

### Instalation

Clone locally and set it as a submodule

```bash
git clone --recursive https://github.com/naver/dust3r
cd dust3r
pip install -r requirements.txt
```

```git
[submodule "dust3r"]
	path = dust3r
	url = https://github.com/naver/dust3r
```

## Fast3r

### Instalation

Clone locally and set it as a submodule

```bash
git clone https://github.com/facebookresearch/fast3r
cd fast3r
pip install -r requirements.txt
pip install -e .
```

```git
[submodule "fast3r"]
	path = fast3r
	url = https://github.com/facebookresearch/fast3r
```
